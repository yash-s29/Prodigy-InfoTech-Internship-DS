# Task 01: Population Distribution Analysis

Problem Statement: Create a bar chart or histogram to visualize the distribution of a categorical or continuous variable, such as the distribution of ages or genders in a population.

Objective: Understand the demographic spread within a dataset.

Key Visuals: Age distribution histograms and gender-based bar charts.

Insights: Identified predominant age groups and gender ratios.

# Task 02: Titanic EDA & Data Cleaning

Problem Statement: Perform data cleaning and exploratory data analysis (EDA) on the Titanic dataset from Kaggle. Explore the relationships between variables and identify patterns.

Objective: Clean messy data and find correlations between features like Class, Sex, Age, and Survival.

Steps: Handled missing values (Age/Cabin), feature engineering, and survival rate analysis.

Findings: Higher survival rates observed among females and first-class passengers.

# Task 04: Social Media Sentiment Analysis

Problem Statement: Analyze and visualize sentiment patterns in social media data to understand public opinion towards specific topics or brands.

Objective: Categorize social media posts into Positive, Negative, and Neutral sentiments.

Approach: Text preprocessing (Natural Language Processing) followed by sentiment scoring and word cloud visualization.

Visuals: Sentiment distribution pie charts and keyword word clouds.

Objective: Visualize accident hotspots and contributing factors to improve road safety.
